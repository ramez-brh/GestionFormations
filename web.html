📖 Présentation de la formation

Le Développement Web est l'une des compétences les plus demandées sur le marché de l'emploi numérique. Cette formation vous enseigne toutes les technologies nécessaires pour créer des applications web complètes, du frontend au backend, en passant par les bases de données.

Accessible aux débutants complets, cette formation suit une progression pédagogique claire avec des projets concrets à chaque étape : site vitrine, blog, e-commerce, application web full-stack.

📚 Programme — 14 Modules

  • 01
    Introduction au Web
    Fonctionnement d'internet, navigateurs, éditeurs de code
  • 02
    HTML5 — Structure
    Balises sémantiques, formulaires, tableaux, multimédia
  • 03
    CSS3 — Mise en page
    Sélecteurs, Flexbox, Grid, animations, transitions
  • 04
    Design Responsive
    Media queries, mobile-first, Bootstrap 5
  • 05
    JavaScript fondamentaux
    Variables, fonctions, conditions, boucles, DOM
  • 06
    JavaScript avancé
    ES6+, promesses, async/await, API Fetch
  • 07
    React.js
    Composants, hooks, props, state, routing
  • 08
    Gestion d'état (Redux)
    Redux Toolkit, Context API, gestion globale
  • 09
    Node.js & Express
    Serveur web, REST API, middleware, routing
  • 10
    Bases de données
    MySQL, MongoDB, requêtes CRUD, Mongoose
  • 11
    Authentification & Sécurité
    JWT, sessions, bcrypt, protection CSRF/XSS
  • 12
    PHP & WordPress
    PHP orienté objet, création de thèmes WordPress
  • 13
    Déploiement & DevOps
    Git, GitHub, Vercel, Netlify, Docker basique
  • 14
    Projet final Full-Stack
    Application complète avec React + Node + MongoDB
